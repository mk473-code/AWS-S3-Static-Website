# AWS S3 Static Website Hosting

## Project Overview
This project demonstrates hosting a static portfolio website using Amazon S3 Static Website Hosting.

## Technologies Used
- HTML
- CSS
- JavaScript
- AWS S3

## Features
- Responsive portfolio website
- Static website deployment
- Public website access
- Cloud hosting using Amazon S3

## AWS Services Used
- Amazon S3

## Project Workflow
1. Created an S3 bucket.
2. Enabled Static Website Hosting.
3. Uploaded website files.
4. Configured bucket policy.
5. Accessed the website through the S3 endpoint.
